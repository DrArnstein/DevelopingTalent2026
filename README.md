# Developing Talent 2026

A small repository collecting proposals, supporting documents, and examples for the "Developing Talent 2026" project.

## Overview

This repo holds draft and final proposals, technical reviews, and supporting materials used while preparing the Developing Talent 2026 project and related funding submissions.

Key files (root):

- `Javits proposal FINAL submission  04-11-2022.pdf` — previous submission
- `Proposal_June17-2026v3.docx` — proposal draft (June 17 2026)
- `Proposal_June18-2026-NEW.docx` — proposal draft (June 18 2026) new
- `Proposal_June18-2026-OLD.docx` — proposal draft (June 18 2026) old
- `Technical Review_S206A220038 - Purdue University TRF.pdf` — technical review
- `fy-2026-84206a-javits-application-notice-and-instructions-113864.pdf` — application instructions
- `index.html` — repository landing page

## Quick start

- To preview the project landing page open `index.html` in a browser or view the file on GitHub.
- The proposals and PDFs are stored at the repository root — click a filename in the repository to download or preview.
