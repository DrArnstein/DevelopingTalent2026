# DevelopingTalent2026

Resources, curriculum and examples for the "Developing Talent 2026" project.

## About
This repository collects materials, lesson plans, exercises, and reference code to support programs and workshops aimed at developing technical and professional talent in 2026 and beyond.

## Contents
- index.html — Project landing page
- README.md — This document
- docs/ — (planned) supporting documentation and lesson materials
- examples/ — (planned) code examples and sample projects

## Getting started
1. Clone the repository:

   git clone https://github.com/DrArnstein/DevelopingTalent2026.git

2. Open `index.html` in a browser to view the project landing page.

3. Add resources in the `docs/` or `examples/` folders as lessons and materials are created.

## Contributing
Contributions are welcome. Suggested workflow:

- Fork the repository
- Create a topic branch for your changes
- Add or update materials in `docs/`, `examples/`, or top-level files
- Submit a pull request describing the changes and which lesson or module they're for
