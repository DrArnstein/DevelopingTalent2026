# DevelopingTalent2026 Project Context

## Purpose

This repository supports development of the FY 2026 Jacob K. Javits Gifted and Talented Students Education Program (Developing Talent Competition) proposal:

Developing Talent Through Psychosocial Strengths: Pathways to Advanced Learning

This file serves as the authoritative project context document for proposal development, document organization, content retrieval, and future collaboration.

## Core Proposal Development Rules

**Rule 1**

Follow FY2026 Javits guidance before all other sources.

**Rule 2**

Use the funded 2022 proposal as the primary structural model.

**Rule 3**

Treat Proposal_June18-2026-NEW.docx as the authoritative working proposal.

**Rule 4**

Treat Proposal_June18-2026-OLD.docx as a source repository of content.

**Rule 5**

Before moving content from OLD to NEW:

1. Identify the source location.
2. Determine the correct FY2026 scoring section.
3. Verify alignment with selection criteria.
4. Revise as needed.
5. Insert into the NEW proposal.

**Rule 6**

Organize content according to reviewer scoring logic rather than historical drafting order.

**Rule 7**

Favor clarity, scoring alignment, and proposal coherence over preservation of original wording.

⸻
